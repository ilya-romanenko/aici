# Архив документации

В этой директории лежат исторические документы, которые больше не являются рабочими требованиями.

## Правила архива

- Архивные файлы не редактируются как источник текущих решений.
- Новые рабочие задачи должны ссылаться на актуальные файлы из корня `docs/`, `docs/guides/`, `docs/deployment/`, `docs/architecture/`.
- Если нужен контекст из архива, ссылка должна быть явной с пометкой `historical`.

## Текущее содержимое

- `roadmaps/` — предыдущие версии roadmap (v1-v7), сохранены для истории решений.
