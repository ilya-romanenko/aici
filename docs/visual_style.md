# Цветовой код (палитра для лендинга)

Основная идея: **технологичность + доверие + премиум**.

* **Главный цвет (акцент)**: глубокий **синий / индиго (#1A237E)** — ассоциация с надёжностью и финансами.
* **Вторичный (акцент)**: неоновый **бирюзовый (#00E5FF)** — символизирует технологичность, AI и свежесть.
* **Фоновые**: тёмный **графит (#121212)** или светлый **серо-белый (#F5F7FA)** (две версии лендинга: тёмная и светлая).
* **Текстовые**:

  * заголовки — белый / почти чёрный (#FFFFFF / #0D0D0D),
  * вторичный текст — серый (#9E9E9E).
* **Успех / рост**: зелёный (#4CAF50).
* **Риск / предупреждение**: красный (#FF5252), но использовать минимально.

Таким образом получится сочетание:
синий (стабильность) + бирюзовый (инновации) + белый/чёрный (контраст) + немного зелёного (рост).
